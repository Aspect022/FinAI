# FinAI ML Service

This directory contains the machine learning models and API for the FinAI project.

## Overview

This service is responsible for:

- Analyzing financial data and news sentiment.
- Predicting market trends.
- Generating personalized investment recommendations.

It exposes a FastAPI endpoint to provide these insights to the main application.

## Getting Started

1. Install dependencies: `pip install -r requirements.txt`
2. Run the API: `uvicorn app.main:app --reload`
